# Learn Playwright Python Project

Welcome to your Playwright Python learning project! This repository contains two scripts that demonstrate how to construct web scrapers using [Playwright](https://playwright.dev/python/). 

This project uses `uv` for lightning-fast Python package management.

## Project Structure

- `basic_sync_scraper.py`: A script that synchronously scrapes `https://quotes.toscrape.com/`. It extracts text content and lists of elements.
- `advanced_async_scraper.py`: An asynchronous script that scrapes `https://books.toscrape.com/`. It demonstrates advanced features like taking screenshots, setting fake authorization headers, and intercepting network requests to block images.
- `pyproject.toml` and `.python-version`: Automatically generated by `uv` to manage dependencies.

## Prerequisites

Make sure you have `uv` installed. If not, see the [uv installation guide](https://docs.astral.sh/uv/getting-started/installation/).

## Setup Instructions

If you haven't already, the project was initialized using:
```bash
uv init .
uv add playwright
uv run playwright install
```

## Running the Scrapers

You can run the scripts using `uv run`, which automatically uses the virtual environment set up for this project.

### 1. Basic Synchronous Scraper
Run the synchronous quotes scraper:
```bash
uv run python basic_sync_scraper.py
```
This will print its progress to the terminal and output the results into a file named `quotes_output.json`.

### 2. Advanced Asynchronous Scraper
Run the advanced books scraper:
```bash
uv run python advanced_async_scraper.py
```
This will demonstrate some of Playwright's powerful features:
- It outputs the scraped data into `books_output.json`.
- It takes a full-page screenshot (`books_homepage.png`).
- It takes an element-specific screenshot (`first_book_element.png`).
- It blocks all images from loading on the page to save bandwidth (you can observe the network interception functionality in the code).
